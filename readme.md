## COMANDOS GIT

- git clone <enlace-repo> -> Clona un proyecto del repo a la computadora.
- git pull -> Actualiza una rama en la pc si recibio cambios en el repositorio.
- git checkout <nombre-rama> -> Moverte a la rama <nombre-rama>
